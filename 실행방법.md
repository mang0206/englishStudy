## 실행 방법

### 사전 요구사항
- Docker, Docker Compose
- Gemini API Key ([Google AI Studio](https://aistudio.google.com/apikey)에서 발급)

### 환경변수

```bash
# .env 파일 열어서 GEMINI_API_KEY 입력
```

### 실행

```bash
docker compose up -d --build

docker compose up -d --build app
```

http://localhost:8080 접속

```bash
# 종료
docker compose down

# 데이터 볼륨 포함 전체 삭제
docker compose down -v
```

postgres 접속
```
docker exec -it et-db psql -U postgres -d english_tracker
```

DB, ES 초기화
```
docker exec -it et-db psql -U postgres -d english_tracker -c "TRUNCATE TABLE expansion_practices, selected_sentences, script_sentences, scripts, task_checks, daily_records RESTART IDENTITY CASCADE;"

curl -X DELETE "http://localhost:9200/selected_sentences"
```
