## 실행 방법

### 사전 요구사항
- Docker, Docker Compose
- Gemini API Key ([Google AI Studio](https://aistudio.google.com/apikey)에서 발급)

### 환경변수

```bash
# .env 파일 열어서 GEMINI_API_KEY 입력
```

### 실행

```bash
docker compose up -d --build
```

http://localhost:8080 접속

```bash
# 종료
docker compose down

# 데이터 볼륨 포함 전체 삭제
docker compose down -v
```
